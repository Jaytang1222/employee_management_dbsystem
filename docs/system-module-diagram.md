## 📋 功能模块详细说明

### **1. 用户认证模块**

**功能描述**：提供系统登录认证功能，保证系统安全性

**核心功能**：
- 用户登录：验证用户名和密码，返回JWT Token
- 用户登出：清除登录状态
- 获取用户信息：根据Token获取当前登录用户信息

**技术实现**：
- JWT Token认证
- BCrypt密码加密
- 登录拦截器

---

### **2. 员工管理模块**

**功能描述**：管理员工基本信息和员工在组织中的分配关系

**2.1 员工信息管理**
- **新增员工**：录入员工基本信息（姓名、身份证、联系方式等）
- **编辑员工**：修改员工信息
- **删除员工**：软删除，修改status为deleted
- **查询员工列表**：支持分页、按姓名/部门/岗位搜索
- **查看员工详情**：显示员工完整信息及关联数据

**2.2 员工分配管理**
- **员工部门分配**：将员工分配到部门（支持主部门+兼职部门）
- **员工岗位分配**：为员工分配岗位
- **员工调动记录**：记录员工调动历史
- **查询员工历史分配**：查看员工的历史部门和岗位

**数据验证**：
- 身份证号：18位格式验证
- 手机号：11位格式验证
- 邮箱：邮箱格式验证
- 年龄：18-65岁范围验证

---

### **3. 组织架构模块**

**功能描述**：管理公司的组织架构，包括部门和岗位

**3.1 部门管理**
- **新增部门**：创建新部门，指定上级部门
- **编辑部门**：修改部门信息
- **删除部门**：检查是否有子部门和员工，软删除
- **查询部门列表**：显示所有部门
- **部门树形展示**：★特色功能，递归展示部门层级结构
- **设置部门经理**：为部门指定经理

**3.2 岗位管理**
- **新增岗位**：创建新岗位，设置基本工资和级别
- **编辑岗位**：修改岗位信息
- **删除岗位**：检查是否有员工在该岗位，软删除
- **查询岗位列表**：显示所有岗位
- **查看岗位详情**：显示岗位详细信息

**技术实现**：
- 部门树：使用递归CTE或递归查询实现
- 支持3-5级部门层级

---

### **4. 考勤管理模块**

**功能描述**：记录和管理员工每日考勤情况

**4.1 考勤记录管理**
- **新增考勤记录**：记录员工签到签退时间
- **编辑考勤记录**：修改考勤信息
- **删除考勤记录**：删除错误的考勤记录
- **查询考勤列表**：按员工、日期、状态筛选
- **自动计算**：自动计算迟到、早退、加班时间

**4.2 考勤统计**
- **员工考勤月报**：★复杂查询，统计员工月度考勤情况
  - 出勤天数
  - 迟到次数和总分钟数
  - 早退次数和总分钟数
  - 加班总小时数
  - 缺勤天数
- **部门考勤统计**：统计部门整体考勤率
- **考勤异常统计**：统计迟到、早退、缺勤情况

**考勤状态**：
- normal：正常
- late：迟到
- early_leave：早退
- absent：缺勤
- leave：请假

---

### **5. 薪资管理模块**

**功能描述**：管理员工薪资发放记录

**5.1 薪资记录管理**
- **新增薪资记录**：录入员工月度薪资
- **编辑薪资记录**：修改薪资信息
- **删除薪资记录**：删除错误的薪资记录
- **查询薪资列表**：按员工、月份筛选
- **自动计算实发工资**：
  ```
  实发工资 = 基本工资 + 津贴 + 奖金 - 扣款 - 社保 - 税
  ```

**5.2 薪资统计**
- **月度薪资汇总**：统计月度薪资总额
- **部门薪资统计**：统计各部门薪资成本
- **薪资成本分析**：分析薪资构成和趋势

---

### **6. 绩效管理模块**

**功能描述**：管理员工绩效考核记录

**6.1 绩效考核管理**
- **新增绩效记录**：录入员工绩效考核结果
- **编辑绩效记录**：修改绩效信息
- **删除绩效记录**：删除错误的绩效记录
- **查询绩效列表**：按员工、日期、等级筛选
- **自动评定等级**：根据分数自动评定等级
  - A：90-100分（优秀）
  - B：80-89分（良好）
  - C：60-79分（合格）
  - D：<60分（待改进）

**6.2 绩效统计**
- **绩效排名**：★复杂查询，按分数排名
- **部门绩效统计**：统计部门平均绩效
- **绩效趋势分析**：分析员工绩效变化趋势

---

### **7. 统计报表模块**（★核心功能）

**功能描述**：提供各类统计报表和数据分析

**7.1 员工统计**
- **按部门统计员工数量**：★复杂查询，统计各部门员工人数
- **按岗位统计员工数量**：统计各岗位员工人数
- **按学历统计员工数量**：统计不同学历员工分布
- **员工入职趋势分析**：分析员工入职趋势

**7.2 考勤统计**
- **员工考勤月报**：详细的月度考勤统计
- **部门考勤率统计**：统计各部门考勤率
- **考勤异常分析**：分析考勤异常情况

**7.3 绩效统计**
- **绩效排名**：员工绩效排名列表
- **绩效分布统计**：统计各等级人数分布
- **绩效等级占比**：计算各等级占比

**技术实现**：
- 使用SQL聚合函数（COUNT、SUM、AVG）
- 使用窗口函数（RANK、ROW_NUMBER）
- 使用GROUP BY分组统计
- 使用JOIN多表关联查询

---

### **8. 数据导出模块**（★核心功能）

**功能描述**：将系统数据导出为Excel文件

**8.1 员工数据导出**
- 导出员工列表（包含部门、岗位信息）
- 支持按条件筛选后导出

**8.2 考勤数据导出**
- 导出指定月份的考勤记录
- 包含考勤统计信息

**8.3 薪资数据导出**
- 导出指定月份的薪资记录
- 包含薪资汇总信息

**技术实现**：
- 使用Apache POI生成Excel文件
- 设置表头样式和数据格式
- 支持大数据量导出

**导出格式**：
- 文件名：`{模块名}_{日期}.xlsx`
- 包含表头、数据、样式
- 自动调整列宽

## 📝 功能特色

### **★ 特色功能**
1. **部门树形展示**：递归展示部门层级结构，支持展开/折叠
2. **按部门统计员工数量**：复杂SQL查询，多表关联统计
3. **员工考勤月报**：详细的月度考勤统计分析
4. **绩效排名**：使用窗口函数实现排名功能
5. **数据导出**：Excel格式导出，支持样式和格式

### **★ 技术亮点**
1. **前后端分离**：Vue3 + Spring Boot
2. **RESTful API**：标准的REST接口设计
3. **JWT认证**：无状态的身份验证
4. **MyBatis**：灵活的SQL映射
5. **软删除**：保留数据，避免误删
6. **数据验证**：前后端双重验证
7. **事务管理**：保证数据一致性
